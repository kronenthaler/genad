programming guidelines:
1. Imports
    a. the order of the imports must be from basic utilities to refinated ones. By instance, 
	import java.io.*;
	import java.util.*;
	import java.awt.*;
	import java.awt.events.*;
	import javax.swing.*;
	  ...
    b. the imports related to this project must be the last, separated by one blank line and sorted in
	the same way described in (a)
    c. if (b) is tie, the packages belong to java go first, org and com come seconds, javax come third.
    d. if (c) doesn't break the tie, use length of the final package name, 
    e. if (d) still tie, use lexicographical order.
